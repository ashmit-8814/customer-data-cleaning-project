# 🧹 Customer Data Cleaning Project

## 📌 Project Overview

This project focuses on cleaning and preparing a messy customer dataset for analysis. Real-world datasets often contain missing values, incorrect formats, and duplicates, which were handled in this project.

## 🎯 Objectives

* Identify data quality issues
* Handle missing values
* Fix incorrect data types
* Remove duplicate records
* Standardize data formats

## 🛠️ Tools Used

* Python (Pandas)
* Jupyter Notebook

## 🔧 Data Cleaning Steps

* Removed duplicate rows
* Converted Age and Salary columns to numeric
* Handled invalid entries (e.g., "abc", "not_available")
* Filled missing values using mean and median
* Standardized date formats
* Filled missing categorical values

## 📊 Final Outcome

* Clean and structured dataset ready for analysis
* Improved data quality and consistency

## 💡 Key Learnings

* Importance of handling real-world messy data
* Data preprocessing techniques
* Practical use of Pandas for cleaning

## 📁 Files

* `analysis.ipynb` → Cleaning process
* `messy_data.csv` → Raw dataset
